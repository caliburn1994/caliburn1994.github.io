## AWS CloudWatch

> Monitor Resources and Applications

作用：

- 收集日志
- 收集指标（metrics ）
- 服务运行状况
- 根据指标或者运行状态，进行自动进行操作

## AWS CloudTrail

> Track User Activity and API Usage

CloudTrail 收集AWS下的所有操作的日志，而 CloudWatch 收集的只是 AWS 部分资源的日志。

## AWS Trusted Advisor

> Optimize Performance and Security

优化 AWS 基础设施，以提高安全性和性能、降低总成本并监控服务限制。优化的内容：

- cost optimization
- security
- fault tolerance
- performance
- service limits
