## 概念

**Managed identities**

通过 **Managed identities** 可以验证 Microsoft Entra-supported 的所有服务。如：通过 Managed identities 可以不使用 key 直接访问 Azure 的数据库服务。

**Service Principal**

application registration, application objects, and service principals in Microsoft Entra ID
