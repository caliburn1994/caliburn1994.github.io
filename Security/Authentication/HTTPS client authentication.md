HTTP客户端验证，比 <u>基本验证</u><sup>Basic Authentication</sup> 和 <u>基于表格的验证</u><sup>Form-based Authentication</sup> 更为安全的验证方式[^1]。

- HTTP客户端验证：通过 <u>带公开密钥的证书</u><sup>Public Key Certificate</sup> 进行验证真实性。
- 基本验证、基于表格的验证：通过验证密码来验证真实性。
