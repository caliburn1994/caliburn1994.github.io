
### 查看文件大小

```bash
$ du -sh *
$ du --summarize --human-readable *
8.2M	Applications
396K	Desktop
6.5G	Documents
2.9G	Downloads

# 按大小顺序，查看文件/文件夹大小
$ du -sh * | sort --human-numeric-sort
$ du -sh * | sort -h
396K	Desktop
8.2M	Applications
2.9G	Downloads
6.5G	Documents
```
