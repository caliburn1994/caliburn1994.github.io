在编程领域，**Parameter**，译为**形式参数**、形参（formal argument），是我们定义函数（或方法）时所用的参数。 <sup>[1][1]</sup>

```python
# 这里的foo参数，就叫Parameter
# 形式上的参数（形参）
def func(foo, bar=None): 
    ...

# 这里的data参数叫Argument
# 实际上的参数（实参）
data = 1 
func(1)
```

[1]: https://docs.python.org/3.9/glossary.html  "Python文档"
