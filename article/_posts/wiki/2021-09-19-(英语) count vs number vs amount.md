## count vs number

作为名词 count 和 number 的意思可互换。[^1]

```bash
the total number of XXX(复数) = XXX count 
```

例如: 

- row count = the total number of rows

## amount vs number

amount 主要用于不可数名词 (mass nouns)，number 通常用于可数名词 (count nouns)。[^2]
