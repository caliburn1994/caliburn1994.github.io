JavaScript里的闭包,demo

```javascript
function foo(){
  var local = 1
  function bar(){
    local++
    return local
  }
  return bar
}

var func = foo()
func()
```

其作用是让local有类似**「成员变量」**的作用, 每次被调用时, 都会改变. 

<br>

问题 : **闭包和Java中类的「成员变量」有什么区别?**

```java
//JAVA代码
class A{
    private  int local = 1;
    
    public int bar(){
        local++;
        return local;
    }
    
}
```

答 : 闭包针对的颗粒度为方法或函数，比Java的颗粒度小，因此影响更小。
